# Moksy.AI Landing Page

A modern, responsive landing page for Moksy.AI — an AI-powered platform that turns your text prompts into beautiful, fully functional websites in seconds.

## 🚀 Features
- Clean, mobile-first responsive design
- AI prompt input for instant website generation
- Gallery of generated website templates
- Feature highlights and FAQ section
- Beautiful, accessible modal for Terms and Conditions
- Built with HTML5, CSS3, Bootstrap 4, and Font Awesome

## 📁 Folder Structure
```
MoksyAILandingPage/
├── index.html
├── README.md
└── src/
    └── assets/
        ├── css/
        │   ├── layout.css
        │   ├── styles.css
        │   └── typography.css
        ├── images/
        │   └── logo.svg
        └── js/
            └── scripts.js
```

## 🛠️ Getting Started
1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd MoksyAILandingPage
   ```
2. **Open `index.html` in your browser.**
   - No build step required; all assets are static.

## ✏️ Customization
- **Gallery Images:**
  - Replace images in `src/assets/images/templates/` with your own screenshots or template previews.
- **Branding:**
  - Update `logo.svg` and meta tags in `index.html`.
- **Styles:**
  - Edit `src/assets/css/styles.css` for custom colors, spacing, or layout tweaks.
- **FAQ & Terms:**
  - Edit the FAQ section and modal content in `index.html` as needed.

## 📱 Mobile Friendly
- The layout is fully responsive and optimized for all devices.
- The gallery section is hidden on mobile for a cleaner experience.
